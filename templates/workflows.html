Advanced forms

Same behavior as run.py commands in the table above.

Populate template from PDF

Runs PDF → QB workbook internally, then fills your Goldman/QB-style template. Optional account id (e.g. 366-3).

Populate template from fields.json

Use fields.json from the “Extract fields” step (or CLI fields) plus your template.

Extract fields (PDF → fields.json)

Builds the QB workbook from the PDF, then emits structured fields for downstream template fill.

Clean extraction JSON

Removes repetitive sections. Optional PDF helps match phrases to pages (same as CLI clean-json with --pdf).

Audit JSON vs PDF (standalone)

Full audit report without re-running extraction. Optional max pages for faster smoke tests.

GitHub & run

See the root README.md for clone, venv, pip install -r requirements.txt, and flask --app app run. Documentation index: docs/README.md.

git clone <your-repo-url>
cd pdf-excel
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
flask --app app run
# Open http://127.0.0.1:5000 and http://127.0.0.1:5000/workflows

Replace the clone URL with your GitHub remote after you push this repository.
